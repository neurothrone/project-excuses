Excuses API Documentation

POST /api/excuses

Create a new excuse.

{
    "category": "school",
    "text": "Aliens abducted my homework."
}
GET /api/excuses

Get all excuses, optionally filtered by category.

GET /api/excuses?category=work
GET /api/excuses/{id}

Get an excuse by its ID.

GET /api/excuses/3
GET /api/excuses/random

Get a random excuse, optionally filtered by category.

GET /api/excuses/random?category=work
GET /api/excuses/categories

Get all excuse categories.

GET /api/excuses/categories
PUT /api/excuses/{id}

Update an excuse by its ID.

{
    "category": "school",
    "text": "Aliens abducted my homework."
}
DELETE /api/excuses/{id}

Delete an excuse by its ID.

DELETE /api/excuses/3
